Создать класс Time для работы с временными промежутками. Интервал должен быть представлен в виде двух полей:
минуты типа int и секунды типа int. При выводе минуты и секунды должны отделяться двоеточием. Реализовать:
-добавление секунд (учесть, что в минуте не должно быть больше 60 секунд)
-вычитание секунд
